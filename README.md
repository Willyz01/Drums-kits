## Features
- 🎵 4 different drum sounds (crash, kick, snare, tom)
- ⌨️ Keyboard controls (C, K, S, T keys)
- 🖱️ Clickable pads with visual feedback
- 🎨 Image background for each drum pad
- ✨ Scale animation on key press
- 📱 Responsive design

## Technologies
- HTML5
- CSS3 (Flexbox, Transitions)
- JavaScript (Vanilla)
- Font Awesome Icons
- Audio files (.mp3)
- Image files (.jfif)
